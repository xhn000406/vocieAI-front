# 资源文件说明

## 必需的文件

请在此目录下放置以下图片文件：

- `icon.png` - 应用图标 (1024x1024 像素)
- `splash.png` - 启动页图片 (1242x2436 像素)
- `adaptive-icon.png` - Android 自适应图标 (1024x1024 像素)
- `favicon.png` - Web 图标 (48x48 像素)
- `notification-icon.png` - 通知图标 (96x96 像素)

## 临时解决方案

如果暂时没有这些文件，可以：

1. 使用在线工具生成占位图片
2. 或修改 `app.json` 暂时移除这些配置
3. 或使用任何图片文件重命名（可能显示不正确，但不影响运行）

## 生成工具

- 图标生成：https://www.appicon.co/
- 启动页生成：https://www.appicon.co/splash
